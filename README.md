mtudetect
---------

Detects the MTU in a network by sending ping-packets of varying size to a host.

Build
-----

Goto folder src and call make.

Installation
------------

This is an OpenWRT-package for GLUON and Freifunk.


Execution
---------

As root execute the following command.

	# ./mtudetect 8.8.8.8

Prints on stdout: 

	1492
